Eigene Podcast Feeds

Created: January 5, 2025
Last Edited: January 6, 2025

Man hat oft das Problem, dass man Podcast-Features (wie viel % von Audio-Lektion/Hörbuch) von Dateien aus dem Internet haben möchte. Z.B. der Morselehrgang von DJ4UF.

Um den Podcast-Player nutzen zu können muss man entweder alle Dateien herunterladen, oder einen Podcast-Feed erstellen. Ich habe einen Podcast-Feed erstellt, da die Flexibilität und Verteilbarkeit besser ist:

Podcast-Feed (Kopiere den Link und füge ihn in deine Podcast-App ein)

Technischer Hintergrund

Siehe im RSS-Standard nach, wie man einen Feed erstellt. Wichtig sind:

channel

  • Title

  • Description

  • Image

item

  • Title

  • Description

  • Enclosure (hier ist der Link zur Datei hinterlegt)

Unvollständiger Code (nur ein Item)
